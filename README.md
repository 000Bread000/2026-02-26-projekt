## 🛠️ Elkészült Feladatok

### 1. 🎓 Iskolai Jegykezelő Rendszer
Egy Python alapú konzolos alkalmazás, amely szétválasztja a tanári és diák jogosultságokat.
- **Funkciók:** Jegyek rögzítése, tantárgyankénti átlagszámítás, dinamikus diákkezelés.
- **Fájl:** `jegykezelo.py`

### 2. ✅ To-Do / Feladatkezelő
Határidő-orientált feladatkezelő rendszer perzisztens adattárolással.
- **Funkciók:** Feladatok hozzáadása/törlése, státuszmódosítás.
- **Adatkezelés:** JSON alapú fájlmentés (adatok megmaradnak újraindítás után is).
- **Fájl:** `todo.py`

### 3. 🎮 "Mastermind" Számkitaláló
Interaktív logikai játék, ahol a felhasználónak egy véletlenszerűen generált számot kell eltalálnia.
- **Funkciók:** Menürendszer, bemeneti validáció (hibakezelés), eredménylista mentése.
- **Fájl:** `jatek.py`

### 4. 🌐 Modern Üzleti Weboldal
Egy "pacek" bemutatkozó oldal egy fiktív technikai kisvállalkozás számára.
- **Technológia:** HTML5, CSS3, JavaScript + Bootstrap 5.
- **Jellemzők:** Teljesen reszponzív (mobilbarát), interaktív kapcsolatfelvételi űrlap, Google Fonts integráció.
- **Fájl:** `index.html`
